# Voiceover Protocols for Documentary

## The Problem

Documentary voiceover is not narration. It's not a filmmaker telling the audience what to think. At its best, voiceover is a second consciousness layered over the image — an inner voice that reveals what the camera cannot show.

Most AI writing assistants produce voiceover that sounds like journalism: explanatory, chronological, authoritative. This is exactly what auteur documentary tries to avoid.

## Framework

### Voice Registers

Define distinct registers the voiceover can operate in. Each register has its own rules.

| Register | Description | When to use |
|---|---|---|
| **Inner monologue** | Subject's thought process, first person | When revealing perception, not facts |
| **Epistolary** | Letter/email voice, addressing someone | When using correspondence as source |
| **Analytical** | Third-person observation, essayistic | When the film needs to contextualize |
| **Poetic** | Fragment, rhythm, repetition | Transitions, emotional peaks |
| **Silence** | No voiceover | When the image is sufficient |

### Rules

1. **Never explain what's visible.** If the image shows it, the voice doesn't say it.
2. **Voiceover contradicts image** is more interesting than voiceover confirms image.
3. **Each register has a texture.** Inner monologue is halting, incomplete. Epistolary is addressed. Analytical is measured. Mixing textures = mixing registers = confusion.
4. **The subject's voice is sovereign.** If the subject has said something in their own words, use their words. Don't paraphrase.
5. **Silence is the default.** Voiceover is added, not removed. Start with no voiceover and justify every addition.

### Protocol per sequence

For each sequence in the edit:

```
1. What does the image communicate alone?
2. What is missing that only voice can add?
3. Which register fits?
4. Write 2-3 lines maximum
5. Read aloud — does it breathe with the image?
6. If the answer to #2 is "nothing" → no voiceover
```

## AI Integration

When using AI to draft voiceover:

- **Provide register explicitly**: "Write in epistolary register, Joshua addressing the reader"
- **Provide image description**: The AI needs to know what's on screen to avoid redundancy
- **Provide source material**: Real quotes, letters, interviews. The AI remixes, it doesn't invent.
- **Max 3 lines per beat**: Force brevity. AI tends toward verbosity.
- **Review against image**: Always test the text against the actual footage. Timing matters.

## References

- Chantal Akerman — voice as intimate confession
- Werner Herzog — voice as unreliable observation
- Terrence Malick — voice as philosophical fragment
- Jean-Luc Godard — voice as political montage
